
Elda säkert! På valborgsmässoafton kommer majbrasorna att tändas runt om i landet för att fira in våren. Här är Scouternas 6 bästa tips på hur du eldar säkert. Dela gärna inlägget!

// brasa=a small, controlled fire used for warmth

1. Elden kräver syre, bränsle och värme (och vatten för att släckas)

// bränsle = fuel

Syre finns runt om oss i luften. Bränsle brukar oftast vara pinnar, kvistar och grenar, men det kan också vara veken och stearinet på ett ljus. Värme brukar komma från friktion, som när tändstickan dras mot plånet på tändsticksasken.

// kvist = twig
// gren = branch
// veken = wick
// stearin = candle

2. Elda på säkra platser

Skydda marken genom att bygga en eldstad med stenar som placeras tätt ihop. Undvik att elda i närheten av torrt (högt) gräs och vid nedhängande grenar. Tänk också på att vinden kan blåsa och föra med sig gnistor och lågor. Ha vatten nära till hands för att kunna släcka om det behövs.

// undvika = to avoid
// gnista = spark
// låga = flame

3. Eld brinner från det lilla till det stora

Förbered ved i olika storlekar och börja med att tända i t.ex. näver (det brinner bra) för att sedan lägga på småkvistar (gärna gran). Lägg sedan på större och större pinnar och grenar allt eftersom.

// ved = wood
// näver = björkbark

4. Njut av elden men ha den under uppsikt.

// njuta = enjoy
// uppsikt = supervision

Använd elden till det som var tanken – laga mat, grilla korv, sprida värme, ljus eller bara ha för mys. Ha alltid elden under uppsikt. Lägg på ved lite då och då, men inte mer än du behöver.

5. Låt elden brinna ut och släck

När elden inte behövs längre är det bäst att låta den brinna ut. Häll sedan försiktigt vatten över glöden tills det inte ryker mer. Häll även försiktigt vatten på de varma stenarna. Men låt dem svalna först, annars kan de spricka.

// hälla = to pour
// försiktig = carefully
// svalna = to cool
// spricka = to fracture, to crack

6. Städa eldplatsen

// städa = to clean

Lägg tillbaka stenarna där du tog dem och städa platsen så att det inte syns att du varit där.
